How It Works
Banker's Algorithm

1. Compute Need = Max − Allocation

2. Initialize Work = Available, Finish[i] = false

3. Find process Pi where:
  • Finish[i] = false
  • Need[i] ≤ Work

4. Work += Allocation[i], Finish[i] = true

5. Repeat until no such process exists

If all Finish[i] = true → Safe State
Otherwise → Unsafe (Deadlock Risk)
